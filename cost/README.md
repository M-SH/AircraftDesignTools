| `casm`                 | `total_doc / (seats × mission_nm) * 100`                | Cost per Available Seat Mile (¢/ASM)                    | Standard airline metric, Doganis 2019                                                                                                                                           |
| `rasm`                 | `revenue / (seats × mission_nm) * 100`                  | Revenue per Available Seat Mile (¢/ASM)                 | Standard airline metric                                                                                                                                                         |
| DOC components         | Fuel, Crew, Maintenance, Navigation, Airport, Ownership | Used for pie & stacked bar charts                       | Derived from above assumptions                                                                                                                                                  |
